# ScoreMarkerManager - Cull Below Player 설정

`cullBelowPlayer`는 **플레이어 Y보다 이만큼 아래로 내려간 마커를 풀에 반환하는 거리**입니다.  
추월한 마커는 즉시 사라지지 않고, 이 거리만큼 멀어질 때까지 화면에 남았다가 그때 제거됩니다.

---

## 값을 30~50으로 둘 때

- **기본값(25)** 대비 **30~50**으로 키우면:
  - 추월한 마커가 **더 오래** 화면에 남습니다.
  - 플레이어가 위로 올라간 뒤에도, 방금 추월한 유저의 닉네임/위치를 **잠깐 더** 볼 수 있어 “방금 누구를 이겼다”는 느낌이 강해집니다.
  - 반대로, **너무 크면**(예: 80~100) 아래쪽에 오래 남은 마커가 많아져 시선이 분산되거나, 같은 마커가 길게 보일 수 있습니다.

- **권장**:  
  - 짧고 강한 피드백을 원하면 **25~30**  
  - “추월한 상대를 조금 더 보여주고 싶다”면 **35~50**  
  - 점프 속도가 빠른 게임이면 **30~40** 정도가 무난합니다.

---

## 요약

| cullBelowPlayer | 효과 |
|-----------------|------|
| 작음 (20~25)    | 추월 직후 마커가 빨리 사라짐. 화면이 깔끔함. |
| 중간 (30~40)    | 추월한 마커가 적당히 남았다가 사라짐. |
| 큼 (45~50)      | 추월한 마커가 더 오래 보임. “방금 이긴 상대” 강조. |

인스펙터에서 **Optimization > Cull Below Player** 값을 위 범위로 바꿔 보면서 체감에 맞게 조정하면 됩니다.
